ผู้สอนลงวันว่าง
ผู้สอน

ผู้สอนแต่ละคนเข้าระบบ → เลือกวันที่พร้อมสอน + รอบเวลา → ระบบบันทึก

📅 เลือกวันว่างเปิดปฏิทินเดือน → กดวันที่พร้อมสอน (เลือกหลายวันได้)
🕐 เลือกรอบเวลาแต่ละวันเลือกได้: 11-13 / 14-16 / 17-19 (เลือกหลายรอบได้)
👤 ข้อมูลผู้สอนชื่อ, ความเชี่ยวชาญ, จำนวนนักเรียนสูงสุดต่อรอบ
✅ ผลลัพธ์ระบบรู้ว่า "วันนี้ มีผู้สอนกี่คน รอบไหนบ้าง" → เปิดให้ลูกค้าจอง

เซลล์สร้างโค้ดส่วนลด (ถ้าต้องการ)
เซลล์

เซลล์สร้างโค้ดโปรโมชั่นส่งให้ลูกค้า เพื่อได้ราคาพิเศษ

🎟️ สร้างโค้ดเซลล์กด "สร้างโค้ด" → ระบบ generate เช่น SALE-A3X9K2
💰 กำหนดส่วนลดลดกี่บาท หรือกี่ % → เช่น ลด 100 บ. หรือ ลด 20%
📋 เงื่อนไขใช้ได้กี่ครั้ง, หมดอายุเมื่อไหร่, ใช้กับคอร์สไหนบ้าง
📱 ส่งให้ลูกค้าส่งโค้ดทาง LINE/โทร → ลูกค้าใส่ตอนจองผ่านเว็บ

3
ลูกค้าจองคลาส
ลูกค้า / เซลล์

ลูกค้าจองเองผ่านเว็บ หรือเซลล์จองให้ — เห็นเฉพาะวันที่มีผู้สอน

🌐 จองผ่านเว็บ (ราคาเต็ม)

  • เปิดเว็บ → เลือกคลาส CPR
  • เห็นปฏิทิน: เฉพาะวันที่มีผู้สอน
  • เลือกวัน + รอบเวลา
  • เลือกแพ็คเกจ Flex ฿600 / Fixed ฿500
  • ใส่โค้ด (ถ้ามี) → ลดราคา
  • ราคาเว็บ = ราคาเต็ม

💼 จองผ่านเซลล์ (ราคาโปร)

  • เซลล์ส่งโค้ด SALE-XXXXXX ให้ลูกค้า
  • ลูกค้าเปิดเว็บ → ใส่โค้ด → ได้ราคาโปร
  • หรือเซลล์จองให้ในระบบหลังบ้าน
  • เซลล์กำหนดส่วนลดได้ตามโปรโมชั่น
  • ราคาเซลล์ = ราคาส่วนลด

📅 ปฏิทินแสดงเฉพาะวันที่เปิดสอน

วันเปิด (สีเขียว)มีผู้สอนลงวัน → ลูกค้าจองได้ → แสดง "เหลือ X ที่"
วันปิด (สีเทา)ไม่มีผู้สอน → กดไม่ได้ → แสดง "ไม่เปิดสอน"
วันเต็ม (สีแดง)มีผู้สอนแต่คนเรียนเต็มแล้ว → แสดง "เต็ม"
วันของฉัน (สีน้ำเงิน)วันที่ลูกค้าจองไว้แล้ว → แสดง "จองแล้ว"

ระบบ

ระบบสรุปทั้งหมด → ลูกค้ายืนยัน → ชำระเงิน → ล็อกที่นั่ง

📋 สรุปการจองคลาส, วัน, เวลา, แพ็คเกจ, ราคาสุทธิ (หลังหักส่วนลด)
⚠️ เงื่อนไขFixed = เลื่อน 1 ครั้ง / Flex = เลื่อนได้ → ลูกค้ากดรับทราบ
💰 ชำระเงินPrompt Pay / โอนธนาคาร / เงินสด / LINE Pay
🔒 ล็อกที่นั่งจ่ายแล้ว → ล็อกที่นั่ง + จับคู่ผู้สอนอัตโนมัติ

ระบบจับคู่ผู้สอนอัตโนมัติ
อัตโนมัติ

เมื่อลูกค้าจองวันไหน → ระบบดูว่าผู้สอนคนไหนลงวันนั้น → มอบหมายอัตโนมัติ

🤖 จับคู่อัตโนมัติวันที่ 20 มี.ค. รอบ 14-16 → ครูเอลงไว้ → มอบหมายครูเอ
⚖️ กระจายงานถ้ามีผู้สอนหลายคนวันเดียวกัน → ระบบกระจายให้คนที่งานน้อยสุด
💰 คำนวณค่าสอน250 บ. × จำนวนนักเรียน (สูงสุด 1,000 บ./วัน)
📧 แจ้งเตือนผู้สอนได้รับแจ้งว่ามีนักเรียนจองวันนั้น
